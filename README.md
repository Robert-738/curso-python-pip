# Proyecto de Juego 

Para correr el juego debes seguir las siguientes intrucciones en la terminal:

```sh
cd game
python3 main.py
```


# Proyecto de App

```sh
git clone
cd app
python3 -m venv env
source env/bin/activate
pip3 install -r requeriments.txt
python3 main.py
```
